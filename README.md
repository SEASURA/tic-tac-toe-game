Tic Tac Toe Game (Python)
📌 Overview

This is a simple command-line Tic Tac Toe game written in Python. The player competes against the computer.

🎮 Features
3x3 Tic Tac Toe board
Player vs Computer gameplay
Input validation
Random computer moves
Win and draw detection
🧠 How It Works
The board is initialized with numbers (1–9)
The computer makes the first move (center position)
The player enters a number to place 'O'
The computer places 'X' randomly
The game continues until:
A player wins
Or the game ends in a draw
▶️ How to Run
Make sure Python is installed
Save the file as tic_tac_toe.py

Run the program:

python tic_tac_toe.py
🎯 Controls
Enter numbers from 1 to 9:
1 | 2 | 3
4 | 5 | 6
7 | 8 | 9
🏆 Winning Conditions

A player wins if they get:

3 in a row (horizontal)
3 in a column (vertical)
3 diagonally
⚠️ Limitations
Computer uses random moves (not AI)
No GUI (console only)
